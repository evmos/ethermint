# Parameters

The `x/feemarket` module contains the following parameters:

| Key                           | Type   | Example     |
| ----------------------------- | ------ | ----------- |
| NoBaseFee                     | bool   | false       |
| BaseFeeChangeDenominator      | uint32 | 8           |
| ElasticityMultiplier          | uint32 | 2           |
| InitialBaseFee                | uint32 | 1000000000  |
| EnableHeight                  | uint32 | 0  |
